# 利用規約（Terms of Service）

本利用規約（以下「本規約」といいます。）は、InstantMesh（以下「本サービス」といいます。）が提供するソフトウェアおよびネットワークインフラストラクチャの利用条件を定めるものです。ユーザーの皆様には、本規約に従って本サービスをご利用いただきます。

---

## 第1条（定義）
1. **「ホスト」**とは、本サービスにアカウント登録を行い、一時的な仮想ネットワーク（以下「ルーム」といいます）を作成・管理するユーザーをいいます。
2. **「ゲスト」**とは、アカウント登録を行うことなく、ホストが発行したインバイトリンクまたはQRコードを介してルームに参加するユーザーをいいます。
3. **「ユーザー」**とは、ホストおよびゲストの総称をいいます。
4. **「接続メタデータ」**とは、通信の内容（ペイロード）を除く、接続に関する情報（ホストのアカウント情報、ルームの作成・解散日時、参加ゲストのIPアドレスおよびニックネーム等）をいいます。

## 第2条（アカウント管理および権限）
1. ホストは、自己の責任においてアカウントおよび認証情報を管理するものとします。
2. ホストは、自身が作成したルームに属するゲストの管理、およびルームの解散に関する全権限と責任を負うものとします。
3. 本サービスは、ホストが作成したルーム内でのトラブル、不正アクセス、データ漏洩等について、一切の責任を負いません。

## 第3条（通信の性質と秘密）
1. 本サービスが提供するVPN通信は、端末間でエンドツーエンド（E2E）で暗号化されます。
2. 本サービスの管理サーバーは、通信の仲介（シグナリング）および暗号化されたパケットの中継（リレー）を行うのみであり、**ユーザー間の通信内容（ペイロード）を復号、閲覧、または保存することはありません**。したがって、通信内容に起因する損害について、本サービスは一切の責任を負いません。
3. 前項にかかわらず、本サービスは、ネットワークの安定運用および不正利用の防止、ならびに法令に基づく対応のため、**通信内容を除く「接続メタデータ」を第4条に定めるところに従い取得・保持します**。
4. 無料で提供されるルームには最大1時間、有料プランでは最大24時間の制限時間が設定されており、時間が経過した場合は事前の通知なく自動的にルームおよび通信経路が消滅します。また、ルーム内でまったく通信が発生しない状態（純アイドル）が30分継続した場合も、同様に自動的に解散されます。これらに伴うデータの不達や作業の中断について、ユーザーは予め了承するものとします。

## 第4条（個人情報および接続メタデータの取扱い）
1. 本サービスは、本サービスの提供にあたり、ゲストのIPアドレスおよびニックネーム、ホストのアカウント情報、ならびにルームの作成・接続に関する接続メタデータを取得します。
2. 本サービスは、取得した接続メタデータを、次の目的の範囲で利用します。
   1. ネットワークの安定運用および品質改善。
   2. 不正利用・悪用（第5条の禁止事項に該当する行為等）の検知および防止。
   3. 法令に基づく開示請求その他の正当な要請への対応。
3. 接続メタデータの保持期間は、原則として**3ヶ月**とします（関連法令等に応じて変更する場合があります）。**通信内容（ペイロード）は取得・保存しません**。
4. 本サービスは、保持する接続メタデータを暗号化のうえ、アクセス権限を最小限の管理者に限定して管理します。また、当該情報は日本国内（東京リージョン）で保管し、国外への移転は行いません。
5. 本サービスは、法令に基づく正当な開示請求があった場合、保有する接続メタデータを関係機関に開示することがあります。
6. ゲストは、ルームへの参加をもって、本条に定める接続メタデータの取得・保持・利用に同意したものとみなします。

ユーザーは、本サービスの利用にあたり、以下の行為を行ってはなりません。
1. 法令または公序良俗に違反する行為。
2. 犯罪行為に関連する行為、またはサイバー攻撃（不正アクセス、DoS攻撃等）の踏み台として利用する行為。
3. 著作権、商標権等の知的財産権を侵害するデータ（海賊版コンテンツ等）を共有・配布する行為。
4. 本サービスのサーバーまたはネットワークの機能を破壊したり、妨害したりする行為。
5. その他、運営者が不適切と判断する行為。

1. 本サービスは、運営コストの最適化およびネットワークの安定運用のために、特定の通信（特にリレーサーバーを経由する通信）に対して、データ転送量の上限（例：1接続あたり100MB）を設定する権利を有します。
